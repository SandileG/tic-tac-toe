# Tic-Tac-Toe (Web Version)

This is a web-based Tic-Tac-Toe game that can be played against another person or an AI.

## How to Play

1. Clone the repository.
2. Open the `index.html` file in a web browser to start the game.

## Features

- Play against another person or an AI.
- AI uses the Minimax algorithm for optimal moves.
- Simple and intuitive web interface built with HTML, CSS, and JavaScript.

## Requirements

- A modern web browser (Chrome, Firefox, Safari, Edge).

## Running the Game

### Locally:
1. Download or clone the repository.
2. Navigate to the folder where the files are located.
3. Open the `index.html` file with any web browser.
