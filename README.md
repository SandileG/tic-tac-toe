# Tic-Tac-Toe

This is a simple Tic-Tac-Toe game that can be played against another person or an AI.

## How to Play

Run the `tic_tac_toe.py` script to start the game.

## Features

- Play against another person or an AI.
- AI uses the Minimax algorithm for optimal moves.
- Game statistics and replay option.

## Requirements

- Python 3.x

## Running the Game

```bash
python tic_tac_toe.py
