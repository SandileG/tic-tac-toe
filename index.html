About the Game

Tic-Tac-Toe is a simple and engaging game where you can challenge yourself or your friends. It's a perfect blend of strategy and fun. Whether you're a novice or a seasoned player, this game is for everyone!

Key Features

AI-Powered Gameplay

Challenge a highly skilled AI opponent powered by the Minimax algorithm, ensuring optimal gameplay every time.

Simple & Intuitive Interface

Play with ease using a minimalist and intuitive interface, designed for smooth, quick matches.

Reset and Replay

Easily reset and replay the game to improve your skills and strategies, with a single click of the reset button.

Game Features

About This Project

This project was inspired by my love for classic games and my desire to challenge myself by creating an AI that can optimally play Tic-Tac-Toe. The goal was to build something fun yet educational, where players can either compete with each other or against a computer opponent.

It also serves as a Portfolio Project for ALX, showcasing my skills in HTML, CSS, and JavaScript, along with the use of algorithms like Minimax. The project was built over a period of a few weeks, refining gameplay mechanics and user experience.

If you'd like to check out the source code, you can visit the GitHub repository.
